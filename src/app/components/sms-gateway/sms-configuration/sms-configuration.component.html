SMS Configuration

If is active checkbox Disable then SMS will be deliver from PROCTUR sender id and with same brand name.

If is active checkbox Enable then take all the details from Proctur side or raise a ticket for username and password.

As per Telecom Regulatory TRAI guidelines, for sending SMS, each SMS has to be registered with DTL platform. We have been observing that some of our clients are not able to send SMS. To facilitate extended service to Proctur’s clients, We have come up with option 1 as an extended process. Option 1- For immediate sending SMS, clients can use Sender Id as “PRCTUR”, but in this case brand name will be Proctur, for each SMS.
