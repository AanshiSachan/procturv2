Release Note

28th Feb 2020
  1. Exception report setting - (Rename Attendance/Exam reminder)
    • Admin can get updates of Attendance and Exam related notification on multiple numbers by selecting Other option.
      Steps: Settings -> Attendance/Exam Notification Settings
  2. Remarks option while taking Attendance
    • Remarks field added while marking Attendance and it helps to notify remarks to Students/Parents. Steps: Home-> Mark Attendance-> (Add Remarks) Mark Attendance
  3. Assign STD/Master Course at a time
    • Assign Standard/Assign Master Course option added in Course and Batch Module to assign these in bulk to selected students.
      Steps: Home->Students->Select Students->Click on option selected students ->Assign Standard (Course Module) or Assign Master Course (Batch Module).
  4. Edit Option Class/Exam Planner
    • Edit option added in Course Planner- To edit details of Description & Homework.
      Steps: Course-> Class Planner->Click on Go (by Selecting Filters)-Edit
  5. Proctur Live Class updates
    • Multiple Layout- Fully control the layout of your virtual classroom, live stream, webinar or webcast, work together on shared notes.
    • Safari support- Users can now also connect via Safari.
    • Improved breakout room statistics Live Class has also improved breakout room usage statistics and worked on general usability of some features so everyone can make the most out of Proctur Live Class.
    • No extension required in chrome for screen share Chrome no longer requires an extension for screen share and the join experience has been made more intuitive. It is available to anyone having chrome version 72 or higher.
  6. Country/State/City/Area Mapping
    • Country/State/City/Area Mapping has been added in Lead and Student section, which can be used while adding an Enquiry and Students.
      Steps: Home Page-> Setup-> City Area Map
  7. Performance Improvement (Student Section)
    • Response time has improved while student data loading.
  8. PTM Enhancement
    • Scheduled PTM details added in PTM home page which can be used to find today's and future PTM schedule as well notify to them.
    • Update and Notify button added on PTM page.
  9. Student's Birthday Details
    • Student Birthday details added on home page, user can check today's Birthday details at the bottom of home page.
  10. Whatsapp Link Integration
    • Whatsapp icon added in global search while searching Enquiry/Student from home page.
31st Jan 2020
  1. Exam Dashboard: Course/Batch -> Reports -> Exam Dashboard
    • Allows you to analyse exam data at various levels: Course/Batch level, Subject level, Date wise, Faculty wise, and Single Exam.
      Steps: Course/Batch -> Reports -> Exam Dashboard
  2. Standard and Subject data is being displayed on Inventory page.
  3. With File Manager now you can share audio file(mp3 and wav) as well.
  4. In Settings low SMS balance reminder added where you can set threshold value on SMS count to get SMS balance notification.
