Release Note

10th July 2020
  1. Study material new UI with improvements on admin web.
  2. Study material with updates on student web.
  3. New settings introduced on My account page.
    • Institute Fee receipt Logo Upload / URL
    • Accounts Email Sender ID
    • Notification Email Sender ID
    • Student ID (Manual/Automatic)
    • Live Class Recorded Session Download Visibility (Setting based)
    • Live Class Recorded Session View Visibility (Setting based)
    • Share app URL for student app and web
    • Feedback URL for student app and web.
      • Privacy Policy Link
      • Terms & Conditions Link
  4. File Manager UI improvements.
  5. Minor bug fixes
3rd July 2020
  1. Registered user changes in estore:
    In this release we made changes on the registered user page. This is a sub menu of estore. On this page, admin gets all the information regarding open users and their products purchased.
    • Admin will get the users according to e-course or product filter. Or they will get all users by clicking on the search button.
    • They will download these details in the excel file as well.
    • Admin can convert the users into enquiry or students.
    • Converted users status will be shown as enquiry or student according to their conversion.
    • Once the user converted into enquiry or student then the action button will be disabled.
    • Search box is also available to search for particular users.
  2. Send notification to open users:
    From now, Admin can send messages and emails to open app users by using the Add/Send notification feature. We have added this option next to All Alumni students on add/send notification pop up. If the user selects the “All open app users” option then all open app users list will be displayed. The functionality is the same as for others.
  3. File manager - faculty account changes
    We have provided the option to download and delete files from the faculty account. The faculty gets the access to delete and download the files which are uploaded by him. But not able to delete the files uploaded by other users (admin/another faculty).
  4. Added the feature
    • Delete file from file manager: Users will be able to delete multiple files at a time. Once a user selects a file, the delete file button will be visible on the file manager page.
    • Edit file name option is available only for Youtube videos, rest of the files will not be able to edit. Delete folder option is removed.
  5. SMS sender id field is added on My Account page
    Sms sender id field is added on My Accounts page. Institute can give sender id according to themself after getting approval from DLT registration. Either message will be delivered by our default sender id.
June 2020
  1. Single device login
    If the setting is enabled then students or open app users are not able to login in multiple tabs at a time. They can log in from the device at a time. If they try to login in multiple tabs or apps, then the warning pop up will be displayed and ask permission to continue a session from this tab. If a student hits yes button then he will log in as a student account in the current session and logout from the previous session (tab). If a student hits the No button then he is active on the previous session (tab). Not login in to the current session. The same thing happens in mobile app)
    • Admin will get the users according to e-course or product filter. Or they will get all users by clicking on the search button.
    • They will download these details in the excel file as well.
    • Admin can convert the users into enquiry or students.
    • Converted users status will be shown as enquiry or student according to their conversion.
    • Once the user converted into enquiry or student then the action button will be disabled.
    • Search box is also available to search for particular users.
  2. Login page performance improvement
  3. Zoom integration
    Admin can add zoom account details in the institute account at his side. This is a setting based feature. It is displayed in my accounts page. Users can add multiple zoom accounts according to host number.
  4. File Manager changes in Faculty Login
    • In a faculty account, share and download option will not be visible to faculty
    • This is setting based option
    • This setting is available in My account - File Manager(Faculty)
    • If this setting is enabled then in the faculty account, faculty can see all the uploaded files by him and other users in a file manager.
    • If this setting is disabled then in the faculty account, faculty can see only those files which are uploaded by him.
  5. Course list ui changes
    Course menu > Total students column and Course expiry date columns are added
  6. Study Material changes in admin account:
    We have added new changes in study material in the admin account. Users can update the file name in study material. Previously we have given this option in file manager, now it is available in this section in admin account. We added the Read Only option in study material while uploading files. This option is only for following categories - Assignment, EBook, notes and previous year question paper. For each file we have given edit, delete and download option except Youtube videos (It has two options-edit and delete)
  7. Dynamic watermark setting added in video setting of study material
    We have added dynamic watermark in videos. In the video setting, there is a setting for dynamic watermark. 3 options added in it - name, email id and contact number. If the user selects all these options then these options will be displayed in video till end. If setting is disabled then these options will not be displayed in video. Setting impact will be displayed in existing videos as well.
  8. Global search Improvement
    Users can search the students by using student id, email id and parent contact number.
May 2020
  1. Live class- setting for disabling notification
    Previously, each and every student joining and leaving notification were displayed on the live class screen. It was very annoying to students to concentrate on their class (studies). Faculty will be able to see these notifications from there side)
  2. Restriction of Youtube URLsharing on student portal
    (From now students won’t be able to share Youtube videos. It is restricted in our application to share Youtube videos from student portal)
  3. Manage product page UI is improved
    • Proper page alignment set on each page.
    • Used the whole page to show more products on it.
    • Removed horizontal scroll bar
    • Proper alignment is given to filters
    • Save and cancel button UI changed
    • Thumbnail alignment is given on review page
    • Image size info is showing on review page which tells user to upload the image in given format.
  4. E-store changes
    We have updated the process of product creation. In this change, admin can create the products according to users i.e. he can create products for students, open app users, both and none.
  5. New Student app has been launched
    Having features of File manager, Study material, Live class, Online exams, Zoom integration
April 2020
  1. Setting based option to send website link to students for login
    (Now it has been made a setting based option to send the website link (Proctur web URL- web.abc.com and in case, virtual hosting is done- then client’s web URL) to students along with login credentials
  2. Download report card more than 9 subjects till 15 subjects
  3. Download summary report
    Setting based: In this feature, we have provided 3 options to download the summary report, i.e. Enquiry, Admission, and fees. Users can download the summary report by using a filter
  4. URL option
    On My Account page, We have added URL fields. Users can enter their Facebook URL, website URL, Youtube URL, WhatsApp URL, Instagram URL, LinkedIn URL, and question and answer URL. Users can get all URL details on this page. ) which will be viewed in the student app/web side menu.
  5. Student profile download
    Users can download student profile from the student page.
  6. Share Youtube URLfrom file manager.
    New field added while uploading documents from the file manager. Users can share Youtube URL from File manager.
  7. Online and offline options on e-course mapping page.
    New options in the e-course mapping page. These options will be displayed to users if the course mapping setting is enabled.
  8. Youtube videos will play on the same page of the student portal.
  9. Student Study Material
    Admin can upload the following types of study material with students i.e. existing video, assignment, eBook’s, notes, audio notes (it supports mp3 and Wav files), images, previous year question paper, slides(PowerPoint files), Youtube URL and videos with students. It will be very helpful for the institute as well as students, to do study from home and also users to share all types of documents or notes with students. Students also get notification whatever the file is uploaded in their ecourse. Users also can manage folders for each subject and topic. It will be a good organized way to share study material with students. This is a setting based option, it is available on general setting page i.e. Student study material visibility under the Miscellaneous section. If the setting is enabled then students can see the study material feature in students’ account. There is one setting that is study material upload notification, if the setting is enabled then students will get the notification of the document which is uploaded by users(admin).
  10. Recorded live class upload and delete in Live class
    We have introduced a new feature which is to upload live class and delete the live class. Users can upload the existing live classes or they can download the existing one live class and upload it. Once they upload it, the “View Recording” button will be enabled and the uploaded file will be displayed. This recording is also available for students who are a part of that live class. Users will delete the live class from the list.
  11. Assign To field added in student module
    Assign To feature is added for students also. This is similar to enquiry’s assign to feature. If you select any counselor name from the field then that student will belong to that counselor. If you do not select any value from drop down then it is automatically assigned to admin.
  12. New Student Web portal with UI enhancement and more features.
